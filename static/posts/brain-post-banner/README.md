# Brain Explorer (Web Demo)

This is a lightweight Three.js demo that loads per‑region meshes and lets you click/explode them.

## Expected files
- `web/assets/regions.json`
- `web/assets/meshes/*.glb` (default) or `.ply` / `.obj`

These are generated by `freesurfer_export.py` using `--web-dir web` and `--mesh-format glb`.

If meshes were compressed with meshopt (via `gltfpack`), the loader already supports it.

If the brain looks faceted, re‑export with smoothing:
```bash
.venv/bin/python freesurfer_export.py \
  --subjects-dir ./freesurfer_subjects \
  --subject sgbrain \
  --lut ./FreeSurferColorLUT.txt \
  --out-dir fs_output \
  --mesh-format glb \
  --no-decimate \
  --smooth-iters 10 \
  --compress \
  --gltfpack ./tools/gltfpack \
  --web-dir web
```

## Run locally
From inside `web/`:
```bash
python -m http.server 8000
```
Open `http://localhost:8000`.

## Controls
- Click a region to show its name
- Toggle “Explode” to spread regions apart
- Search to filter visible regions
